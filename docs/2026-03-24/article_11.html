100 AI CLI Agent Commands: Guide to Claude Code, Aider, and Cursor CLI（100 個 AI CLI Agent 指令：Claude Code、Aider 和 Cursor CLI 指南）

原文來源：codeforgeek.com
以下為核心要點摘要，附英文技術詞彙對照（紫色）幫助理解。

核心摘要

這篇文章是一份 AI CLI agents（AI 命令列代理） 的完整指令參考手冊，涵蓋三大主流工具：Claude Code、Aider、Cursor CLI。文章將 AI CLI agents 定位為「住在終端機裡的配對程式設計師」（pair programmer that lives inside your terminal），能理解自然語言需求並直接操作程式碼。

Claude Code 核心指令分類

安裝與設定

# 安裝 Claude Code
npm install -g @anthropic-ai/claude-code

# 登入驗證
claude auth login

# 更新工具
claude update

Session 管理指令

  • /init — 初始化專案，生成 CLAUDE.md
  • /clear — 清除當前對話上下文（context）
  • /compact — 壓縮對話歷史以節省 token（令牌）
  • /save / /load — 儲存/載入對話狀態

程式碼工作流程指令

  • /review — 程式碼審查（code review）
  • /commit — 生成 git commit 訊息並提交
  • /diff — 檢視程式碼變更差異

Aider 核心指令

  • /add / /drop — 加入/移除檔案到上下文
  • /read — 以唯讀模式加入參考檔案
  • /git — 執行 git 指令
  • /undo — 撤銷最後一次 AI 修改

三工具比較

  • Claude Code：最深度的 Anthropic 整合，支援 MCP（模型上下文協定） 和 hooks（鉤子），適合大型專案
  • Aider：開源、多模型支援，以 TDD（測試驅動開發） 工作流著稱
  • Cursor CLI：IDE 功能延伸到終端，適合已在用 Cursor 編輯器的開發者

對程式設計師的意義

這份指令參考表是日常使用 AI coding agents 的實用速查手冊。特別值得注意的是，三大工具在核心工作流上的指令設計趨於標準化（/review、/commit、/diff），學會其中一個後切換到另一個的學習成本相當低。建議收藏作為日常參考。
