[Plugin] Cortex: An AI Obsidian Vault Agent Powered by Claude Code

原文來源：https://forum.obsidian.md/t/plugin-cortex-an-ai-obsidian-vault-agent-powered-by-claude-code/112430
以下為核心要點摘要，附英文技術詞彙對照（紫色）幫助理解。

核心摘要

Cortex 是一個 Obsidian（知識管理筆記工具）外掛程式，將 Claude Code 整合為 vault（筆記庫）內的 AI agent。透過側邊面板進行對話，Claude 可以自主管理筆記，無需另外的 API 金鑰，直接利用現有的 Claude Pro/Max 訂閱。

核心功能

  • 持久對話面板（persistent chat panel）：在 Obsidian 內直接與 Claude 對話
  • 完整 vault 存取權限：讀取、寫入、建立、移動筆記
  • Session 持久化（session persistence）：對話記錄儲存於 .obsidian/plugins/cortex/.claude/sessions/
  • 可設定上下文系統（configurable context system）：Session 開始時自動注入資料夾和檔案樹
  • 跨 session 記憶（cross-session memory）：透過自主管理上下文檔案實現
  • 無需 API 金鑰：透過 CLI（命令列介面）使用現有 Claude 訂閱

安裝方式

  • 目前處於 Beta 階段，透過 BRAT（Beta 審查員和測試員社群外掛工具）安裝
  • 需要已安裝 Claude Code CLI 並登入

對程式設計師的意義

這個外掛程式展示了將 Claude Code 嵌入現有工具的實際模式。對於使用 Obsidian 管理技術筆記、文件或知識庫的開發者，Cortex 提供了一個不需要額外 API 費用的 AI 助理，可以直接理解並操作整個 vault。這也是 Claude Code 作為通用 agent 框架，被嵌入第三方應用的具體範例。
