# 報表系統規格書

> 以xx為主題的學習追蹤報表，分三種角色視角。

---

## 一、整體架構

| 檔案 | 角色 | 說明 |
|------|------|------|
| `report.html` | 學生 | 個人讀書報表 |
| `parent_report.html` | 家長 | 子女總覽，可切換多位子女 |
| `teacher_report.html` | 老師 | 班級總覽，可切換多班 |

---

## 二、共用設計

- **視覺主題**：暗色調（深藍/紫/金）
- **圖表庫**：Chart.js 4.4
- **字型**：PingFang TC / Microsoft JhengHei / Noto Sans TC
- **響應式**：支援手機（600px）與平板（900px）斷點
- **科目對應**：數學、國文、英文、理化、社會

## 三、學生報表（report.html）
https://lastsummer.github.io/teamsAI/ai-god/report.html
**頁籤：學習統計 / 活動與推薦 / 考試倒計時 / 排行榜(PRO)**

### 學習統計
| 元件 | 說明 |
|------|------|
| 統計卡 ×4 | 累計學習時長、學習次數、日均時長、中斷次數 |
| 科目學習分布圖 | 雷達圖，依科目顯示時長佔比 |
| 學習時段分布 | 長條圖，可切換「24小時」或「星期累積」 |
| 學習分布 | 天/週/月/年切換 + 前後日期導航；顯示時長/次數/中斷三種指標 |

### 活動與推薦
| 元件 | 說明 |
|------|------|
| 學習活動日誌 | 月曆，有活動的日期顯示標記，可前後翻月 |
| 近期活動推薦 | 事件列表，含日期、名稱、描述 |
| AI 學習建議 | 聊天介面；每日免費額度限制，超出顯示付費解鎖提示 |

### 考試倒計時
| 元件 | 說明 |
|------|------|
| 倒計時 | 翻牌計時器，顯示距考試剩餘天/時/分/秒 |
| 準備進度 | 進度條，顯示考試準備百分比 |
| 警示橫幅 | 剩餘天數不足時顯示警示 |

### 排行榜（PRO）
| 元件 | 說明 |
|------|------|
| 全校/全服排行榜 | TOP 10，可切換日/週/月/年/總 |
| 線上統計 | 即時顯示學習中人數、一小時內完成人數 |

## 四、家長報表（parent_report.html）
https://lastsummer.github.io/teamsAI/ai-god/parent_report.html
**頁籤：學習統計 / 活動與推薦**

### 子女切換
- 固定在導覽列下方，以 pill 按鈕切換，顯示頭像與名字

### 學習統計
| 元件 | 說明 |
|------|------|
| 子女摘要橫幅 | 顯示頭像、名字、備註標籤、近期學習分（/100） |
| 家長洞察卡 | AI 自動產生的重點提示（如學習異常、建議關注點） |
| 統計卡 ×4 | 同學生端（累計時長/次數/日均/中斷） |
| 科目分布圖 + 時段分布 | 同學生端 |
| 學習分布圖 | 同學生端，依選取子女動態更新 |

### 活動與推薦
| 元件 | 說明 |
|------|------|
| 學習活動日誌 | 月曆（同學生端） |
| 近期活動推薦 | 事件列表（同學生端） |
| AI 學習建議 | 聊天介面（同學生端，含免費額度與付費解鎖） |

## 五、教師報表（teacher_report.html）
https://lastsummer.github.io/teamsAI/ai-god/teacher_report.html
班級學習報表
 
**頁籤：班級學習數據 / 高風險名單**

### 班級切換
- 固定在導覽列下方，支援多班（範例：三年甲/乙/丙班）

### 班級學習數據
| 元件 | 說明 |
|------|------|
| 統計卡 ×5 | 學生人數、平均學習時長、活躍率、平均中斷次數、平均學習分（附上週變化） |
| 學習趨勢圖 | 折線圖，週為單位；可切換「週平均」/「總時長」/「活躍人數」 |
| 科目分布圖 | 圓餅或長條圖，顯示全班各科學習佔比 |
| 熱門活動排行 | 橫向長條圖，顯示各活動參與人數 |
| 教學進度評估 | 進度軌道（適中/略快/偏慢）標示 |
| 學習時段分布 | 全班當日24小時分布折線圖 |
| 班級學習分布直方圖 | 本月平均分（0–100 分段）的人數分布 |

### 高風險名單
| 元件 | 說明 |
|------|------|
| 匯出功能 | 匯出 PDF / Excel（單次計費） |
| 風險摘要卡 ×3 | 放棄學習 / 過度學習 / 觀察中，各類人數 |
| 風險學生清單 | 每位學生顯示：類型標籤、頭像、姓名、座號、學習時長、中斷次數、學習分、風險標籤、AI 建議措施 |

#### 風險分類規則
| 類型 | 顏色 | 觸發條件（範例） |
|------|------|------|
| 放棄學習 | 紅色 | 連續中斷多次、長時間未登入 |
| 過度學習 | 橙色 | 日均學習 > 10 小時、深夜仍在學習 |
| 觀察中 | 紫色 | 時長下滑、活動參與減少但未達高風險 |

| 功能 | 位置 | 說明 |

| AI 學習建議（超額） | 學生/家長 活動與推薦 | 每日免費次數限制，超出需付費解鎖（10次上限） |
